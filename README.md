# 🎮 Tic Tac Toe

A modern, responsive Tic Tac Toe web game built with pure **HTML, CSS, and JavaScript** — no frameworks, no dependencies. Two players take turns marking X and O on a 3x3 grid, with a live scoreboard, smooth animations, and a clean UI.

## ✨ Features

- 🎯 Classic 3x3 Tic Tac Toe gameplay
- 👥 Two-player mode (X and O)
- 🏆 Live scoreboard (Wins for X, O, and Draws)
- 🔄 Restart button to play again instantly
- 🎨 Clean, modern, professional UI
- 📱 Fully responsive — works on mobile, tablet, and desktop
- ✨ Smooth animations (cell pop-in, winning highlight glow)
- 🎉 Automatic winner and draw detection
- 🧠 Beginner-friendly, well-commented code

## 🚀 Getting Started

### Prerequisites
- A web browser (Chrome, Firefox, Edge, etc.)
- **VS Code** (recommended) with the **Live Server** extension

### Installation

1. **Clone or download** this repository:
````bash
   git clone https://github.com/leoxzayn/tic-tac-toe.git
````

2. **Open the folder** in VS Code:

````bash
   cd tic-tac-toe
   code .
````

3. **Run the project**:

   * Right-click `index.html`
   * Select **"Open with Live Server"**
   * The game opens in your browser at `http://127.0.0.1:5500`

> 💡 No Live Server? Just double-click `index.html` to open it directly in your browser.

## 🎮 How to Play

1. Player **X** starts first.
2. Click any empty cell to place your mark.
3. Players alternate turns (X → O → X → O…).
4. First to get **3 in a row** (horizontal, vertical, or diagonal) wins\!
5. If all 9 cells fill without a winner, it's a **Draw**.
6. Click **Restart Game** to clear the board (scoreboard stays).
